API Reference¶

This section documents the core modules of the PC-UDRL package.

Agents¶

class pc_udrl.agents.udrl_agent.UDRLAgent(state_dim, action_dim, discrete, hidden_dim=256)[source]¶

Bases: Module

Upside-Down Reinforcement Learning (UDRL) Agent.

This agent predicts the optimal action given a state and a command (horizon, desired_return). It learns via supervised learning to mimic trajectories that achieved high returns, effectively inverting the RL problem (mapping Command -> Action).

Parameters:

  • state_dim (int) – Dimension of the observation space.

  • action_dim (int) – Dimension of the action space.

  • discrete (bool) – Whether the action space is discrete (True) or continuous (False).

  • hidden_dim (int, optional) – Size of the hidden layers. Defaults to 256.

Pessimists¶

class pc_udrl.pessimists.quantile.QuantileRegressor(state_dim: int, hidden_dim: int = 256, q: float = 0.9, cfg: Any | None = None)[source]¶

Bases: Module

Pessimistic Oracle using Quantile Regression.

Predicts the tau-th quantile of the return distribution for a given state. Used to clamp user commands to a realistic/safe range, implementing the concept of “Pessimistic Projection”.

Parameters:

  • state_dim (int) – Dimension of the observation space.

  • hidden_dim (int, optional) – Size of the hidden layers. Defaults to 256.

  • q (float, optional) – Quantile to predict (e.g., 0.9 for 90th percentile). Defaults to 0.9.

  • cfg (Any, optional) – Configuration object containing device settings.
